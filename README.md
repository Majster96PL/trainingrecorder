# Training Recorder 
**Traning Recorder List**

## Technology
**Java,
Spring Web,
Spring Security,
Spring Data JPA,
MySQL,
Lombok.**

## Installation
- Download the project,
- Create DataBase,
- Change 'ddl-auto' in aplication.properties from  'update' to 'create',
- Run project,
-Using browser to testing

## Data Base MySQL
```mysql
CREATE DATABASE recorder;
```

## Browser

```
localhost:8080/training
```
